# GenAI Chatbot Testing Framework

## 📌 Overview
This project automates testing of chatbot responses using a hybrid AI approach (real API + fallback).

## 🚀 Features
- Hybrid AI client (Gemini + mock fallback)
- Response validation (keyword, meaningful output)
- Hallucination detection
- Data-driven testing using pytest
- Logging for debugging
- HTML reporting

## 🛠 Tech Stack
- Python
- Pytest

## ▶️ How to Run
```bash
pytest -v -s
